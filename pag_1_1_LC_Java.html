Características

Java es el lenguaje de programación con el que muchos progrmadores empiezan (poniendo a lenguajes visuales como Scratch aparte)

Facilidad de aprendizaje Difícil-illa: A pesar de ser el lenguaje con el que más gente empieza a programar como tal, también es uno de los lenguajes que pueden complicarse. Pero con dicha curva de aprendizaje difícil viene el beneficio de saber moldear a tu antojo a Java para crear todo tipo de cosas
Uso Mayoritariamente en el desarrollo de aplicaciones de cualquier tipo, pero eso no excluye sus programas se utilizen como procesadores de texto y hojas de cálculo
Flexibilidad del código Amplia: Java es un lenguaje que se puede meter en todos los aspectos de la informática, como en los juegos (Minecraft), las apps normales y de móvil. Si quieres un lenguaje estable con el que desarrollar tus primeras aplicaciones, ¡Este es un muy buen comienzo!
Entorno de utilización más popular NetBeans y Eclipse

Cómo se escribe un "hola mundo"

Se busca una función dentro de la clase que se llame main y en esta se escribe:

"System.out.println("Hola mundo")"
