Características

C# es un lenguaje de programación orientado a objetos. C# proporciona construcciones de lenguaje para admitir directamente estos conceptos, por lo que se trata de un lenguaje natural en el que crear y usar componentes de software.

Facilidad de aprendizaje Difícil si eres nuevo: Al ser un lenguaje orientado a objetos, es muy recomendable que primero se empiece con otro lenguaje si eres un principiante, como por ejemplo, Python
Uso Popularmente en videojuegos gracias a motores de juego como Unity, pero también puede utilizarse para otras aplicaciones
