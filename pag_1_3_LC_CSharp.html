Características

C# es un lenguaje de programación orientado a objetos. C# proporciona construcciones de lenguaje para admitir directamente estos conceptos, por lo que se trata de un lenguaje natural en el que crear y usar componentes de software.

Facilidad de aprendizaje Difícil si eres nuevo: Al ser un lenguaje orientado a objetos, es muy recomendable que primero se empiece con otro lenguaje si eres un principiante, como por ejemplo, Python
Uso Popularmente en videojuegos gracias a motores como Unity, pero también puede utilizarse para otras aplicaciones
Flexibilidad del código Alta. C# es utilizado en el desarollo de videojuegos, al crear aplicaciones para Windows, y en el desarollo web
Entorno de utilización más popular Microsoft Visual Studio (Un entorno modular el cual se puede personalizar para tus necesidades) y Kite

Cómo se escribe un "hola mundo"

Se busca una función dentro de la clase que se llame Main y en esta se escribe:

Console.WriteLine("Hello World!");
