Características

Python es un lenguaje de programación de código abierto, orientado a objetos, muy simple y fácil de entender. Tiene una sintaxis sencilla que cuenta con una vasta biblioteca de herramientas, que hacen de Python un lenguaje de programación único.

Facilidad de aprendizaje Muy fácil: De hecho, es EL mejor lenguaje para empezar a picar código si eres un completo principiante o si Scratch se te empezó a quedar un poco pequeño debido a su facilidad de lectura y sencilla sintaxis
Uso El uso de Python está presente en el Big Data, en el desarrollo web, en la programación de placas Arduino e incluso en el desarrollo e investigación IAs (Inteligencias Artificiales)
Flexibilidad del código Por ser un lenguaje tan fácil de leer, Python se puede aplicar a muchos casos y problemas
Entorno de utilización más popular Py charm y SPYDER
